我的高中（高一、高二）数学老师命题 ;-)

三个数390, 455, 546的最大公约数是（）

  • A. 65
  • B. 91
  • C. 26
  • D. 13

此题是单选题（以下没有特别说明，均为单选），不用解题过程，我们采用二分：13, 26, 65, 91先看65是否是三个数的最大公约数！546不能除尽，所以65, 91都不满足条件，再看26，455不能除尽。

也可以用穷举：

  for (int i = 2; i < 390; i++) {
      if (390 % i == 0 && 455 % i == 0 && 546 % i == 0)
          printf("%d\n", i);
  }
